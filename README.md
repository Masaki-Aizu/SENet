# SENet
## ポイント
- SENetとは、SEブロックを組み込んだネットワークアーキテクチャの総称
- SEブロックとは、特徴量マップのチャネル間の相互依存性を明示的にモデル化するための構造
- 特徴量マップに、重要度に応じ**アテンション**をかける働きをする
- 既存のアーキテクチャにSEブロックを組み込むことで学習効率、精度が向上する<br>
